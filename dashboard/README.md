# Dashboard

The dashboard is the operator-facing UI for device visibility, command dispatch, and OTA awareness.

## Initial UX Direction

- fleet summary at the top
- device list with firmware and last-seen state
- device detail and command panel
- OTA release visibility
- AI suggestion panel with explicit approval flow

## Suggested Next Steps

- connect to backend API
- add authentication and role-aware navigation
- build device detail route
- add live updates with WebSocket or polling
